# BINARY SEARCH ALGORITHM
Binary Search is a simple <u><span style="color: orange;">searching algorithm</span></u>. It works by repatedely dividing he search interval by half.

#### Step by Step

## Time Complexity
Best Case: O(1)

Average Case: O(log n)

Worst Case: O(log n)
